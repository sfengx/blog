### 项目介绍

本项目为个人博客，使用ruby、jekyll，托管至github

### 本地预览

```
## 博客生成，默认生成再_site目录下，当然也可以在配置文件中自定义
jekyll build
## 开启jekyll本地预览
jekyll server
```
